# Unit One: Part Two - Code Your Own Adventure


### 1.1 Confirm

```sh
confirm("Ready player one?");
```

### 1.2 Old Enough to play

```sh
var age = prompt("What is your age?");

if (age < 13){
    console.log("You may play, however we assume no responsibilty for the content of our game.");
} else {
    console.log("Play on peasant!");
}
```

### 1.3 Add some Story

```sh
console.log("You are at an Arctic Monkeys concert, and you hear the lyrics 'Get on your dancing shoes' ");
```

### 1.4 First Move
```sh
console.log("Suddenly they stop and Alex Turner asks: 'Who wants to have a dance off with me?' ");

userAnswer = prompt("Do you wnat to have a dance off with Alex Turner?");
```

### 1.5 The Story Heats up

```sh
if (userAnswer === "Yes"){
    console.log("Alex starts the dance off , you pull of a sick freeze and the crowd roars, you win!");
} else {
    console.log("Alex Turner shakes his head sings 'So what you scared for?'");
}
```

### 1.6 Asking for feedback

```sh
var feedback = prompt("Please rate my game below from one to five!")l

if (feedback > 3){
    console.log("Thank you, can't wait for the next dance off!");
} else {
    console.log("I'll keep practicing my coding and dancing skills!");
}
```
